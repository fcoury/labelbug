Creating radio_button_tag '<%= @name %>', '<%= @value %>' and label_tag '<%= @id %>', 'Apache 2.2'

Renders like this:

<%= @radio %> <%= @label %>

If you try to click the label, it won't select the radio as expected, because the radio id rendering took out the dot, while the label for rendering did not.

HTML for radio:
<%= h(@radio).gsub(/id="(.*?)"/, '\0') %>

HTML for label:
<%= h(@label).gsub(/for="(.*?)"/, '\0') %>
